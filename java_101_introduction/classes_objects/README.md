# Classes and Objects

Java is an object-oriented programming language. Classes are blueprints for creating objects.

## Class Declaration

```java
public class ClassName {
    // Fields (instance variables)
    // Constructors
    // Methods
}
```

## Key Concepts

- **Class**: A blueprint/template for objects
- **Object**: An instance of a class
- **Fields**: Variables that belong to the class/object
- **Methods**: Functions that belong to the class
- **Constructor**: Special method called when creating objects

## Creating Objects

```java
ClassName objectName = new ClassName();
```

## Accessing Members

```java
objectName.fieldName;        // Access field
objectName.methodName();     // Call method
```

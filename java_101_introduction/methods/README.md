# Methods

Methods (also called functions) are reusable blocks of code that perform specific tasks.

## Method Declaration

```java
returnType methodName(parameterType parameterName) {
    // method body
    return value;  // if returnType is not void
}
```

## Key Concepts

- **Return Type**: The type of value the method returns (`void` if no return)
- **Method Name**: Identifier for the method (follows camelCase convention)
- **Parameters**: Input values passed to the method
- **Method Body**: The code that executes when the method is called
- **Return Statement**: Sends a value back to the caller

## Method Types

- **Void methods**: Don't return a value
- **Value-returning methods**: Return a value of specified type
- **Methods with parameters**: Accept input values
- **Methods without parameters**: No input needed

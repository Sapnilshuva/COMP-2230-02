# Control Flow

Control flow statements allow your program to make decisions and repeat actions.

## If-Else Statements

Use `if`, `else if`, and `else` to execute code based on conditions.

```java
if (condition1) {
    // code if condition1 is true
} else if (condition2) {
    // code if condition2 is true
} else {
    // code if neither condition is true
}
```

## Loops

Loops repeat code execution.

### For Loop
```java
for (initialization; condition; increment) {
    // code to repeat
}
```

### While Loop
```java
while (condition) {
    // code to repeat
}
```

### Do-While Loop
```java
do {
    // code to repeat (executes at least once)
} while (condition);
```
